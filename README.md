# Adaptive Integration
### Ryan Washburne 2019

Built with:
[GatsbyJS](https://www.gatsbyjs.org/),
[Contentful](https://www.contentful.com/), &
[Netlify](https://www.netlify.com/)

#### Getting Started
```
yarn install
yarn start
```

#### Before you go...
`.contentful.json` file is required. Example:
```
{
  "development":{  
     "host": "",
     "spaceId": "",
     "accessToken": ""
  },
  "production": {
    "spaceId": "",
    "accessToken": ""
  }
}
```
